Assignment

JAVA FX Flight Reservation App

For second-year Application Development, we were tasked with creating a flight reservation app utilizing JavaFX. The app is built around specific sections that use Java to organize, call, and edit data.

The Flight class handles flight details, such as where it's going, when, and the cost. It makes sure all this information is correct and makes sense, like ensuring that a flight date is in the future - not six months ago. The Passenger class keeps track of the passengers, including important details like name and passport number, and checks that this information is properly filled out.
